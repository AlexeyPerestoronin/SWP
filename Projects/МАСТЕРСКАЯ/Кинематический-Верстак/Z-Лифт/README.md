# Что улучшить в версии 2
1. «позиционный уголок» должен быть расширен для крепление «кулис-инструментальных» в отдельные отверстия
2. надо избавить от проставочных колец
3. продумать систему отверстий для смены пластин (спроектировать базовые пластины)
   1. процедура смены пластины должна быть максимально удобной (занимать меньше 10 секунд)
   2. должна быть возможность болтовой регулировки по высоте от 0 до 100мм
4. Добавить крепления для газ-лифта (или пружин)
   1. газ-лифт должен иметь регулируемое крепление, позволяющее регулировать преднатяг (в пределах 50мм)
5. Добавить механизм выбора глубины погружения
6. Проработать систему внешнего крепления
    - обеспечить возможность горизонтального крепления
    - обеспечить возможность вертикального крепления
7. Добавить поперечные стабилизаторы
8. Уменьшить Z-перемещение до 200мм и вылет до 250мм

# Что улучшить в версии 3
1. T-образная крестовина
2. 200мм = высота вертикального хода
3. 250мм = длинна горизонтальной кулисы
4. инструментальный ложемент по середине 2 частей
   1. одна часть не более 5 кг веса
   2. боковое крепление инструментария (не фронтальное)
5. болтовая система фиксации положения
6. вырезы под гайку для TIG-сварки

# Особенности сборки:
1. Толщину металла задать отдельной переменной
2. Толщину упорного подшипника задать отдельной переменной
3. Спроектировать подшипник как отдельный проект
4. Спроектировать гайку как отдельный проект

# Что улучшить в 4-ой версии проектирования после воплощения в металле версии-3
1. Если, возможно использовать 1 инверсор, а не 2
2. Расположить инверсор внутри пантографа
3. Необходимо добавить РЖ во всё длину основания + РЖ соединяющие правый и левый пантографы
4. Не делать от ступ (даже 0.125)
5. Все болты предусмотреть заранее и утопить их 6-гранные головки внутрь соответствующей детали
6. Необходима фиксация только в верхнем положении (в нижнем не нужно)
