# Что улучшить в версии 2
1. Улучшить кинематику:
    - «позиционный уголок» должен быть расширен для крепление «кулис-инструментальных» в отдельные отверстия
    - надо избавить от проставочных колец
2. Изменить площадку для инструмента:
    - продумать систему отверстий для смены пластин (спроектировать базовые пластины)
    - процедура смены пластины должна быть максимально удобной (занимать меньше 10 секунд)
    - должна быть возможность болтовой регулировки по высоте от 0 до 100мм
3. Добавить крепления для газ-лифта (или пружин)
    - газ-лифт должен иметь регулируемое крепление, позволяющее регулировать преднатяг (в пределах 50мм)
4. Добавить механизм выбора глубины погружения
5. Проработать систему внешнего крепления
    - обеспечить возможность горизонтального крепления
    - обеспечить возможность вертикального крепления
6. Добавить поперечные стабилизаторы
7. Уменьшить Z-перемещение до 200мм и вылет до 250мм

# Особенности сборки:
1. Толщину металла задать отдельной переменной
2. Толщину упорного подшипника задать отдельной переменной
3. Спроектировать подшипник как отдельный проект
4. Спроектировать гайку как отдельный проект
