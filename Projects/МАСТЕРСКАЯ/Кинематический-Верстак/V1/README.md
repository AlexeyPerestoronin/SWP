# Кинематический верстак
**Задача: необходимо разработать верстах для производства эпоксидных столов!**

# Изначальные требования:
## Требования к основанию:
1. Размер не менее чем 1800 на 1000 мм
2. Необходимо обеспечить возможность боковых пристроек
3. (?) Использовать имеющиеся столешницы размером 1000x600x50 в качестве модулей

## Требования к кинематики:
1. Возможность движения в 3 плоскостях (X, Y, Z)
2. Жёсткая, чтобы был минимальный люфт (< 0.5)
3. Возможность фиксации положения в любой из плоскостей
4. Z-координата должна регулировать при помощи домкрата, расположенного под столешницей
5. Конструкция образующая Y-координату, должна выступать базовой платформой для приспособления ручного электрического инструмента, следующего типа:
    - фрезе
    - лобзик
    - циркулярные пилы под дереву и металлу
    - дрель  
    (возможно, платформы будут разные для каждого из инструментов)

# Улучшить во второй версии:
## Основание:
1. Отсутствие встроенной системы перемещения качения:
    - надо придумать домкратную систему, которая будет позволять приподнимать верстак целиком для перемещения на колёсиках
2. Непродуманные боковые модули:
    - должны регулироваться по высоте
    - должны быть снимаемыми

## Кинематика:
1. Слишком тяжёлая:
   - Y-координата должна состоять из одной вертикальной направляющей с каждой из сторон
   - Z-координата должна находиться над столешницей
   - должна быть дополнительная подпружиненная Z-координата, для обеспечения само-возврата навесного оборудования
2. Не модульная:
    - X-координата должна быть дополнительной частью и поставляться в качестве навеса на каркас
    - Z-координата должна быть дополнительной частью X-координаты
    - Y-координата должна быть дополнительной частью Z-координаты
3. Не защищённая:
    - все направляющие линейного перемещения должны быть развёрнуты вниз либо располагаться вертикально во избежание оседания стружки/опилков

## Столешницей:
1. Слишком сложная:
    - проектировать как отдельный праметрический проект (отдельная часть сборки)
    - крепление к основанию должно производить при помощи системы шпилек-M16 (возможно М20)
2. Монолитная:
    - столешница должна быть модульной, состоящей из 2, 3 или 4 частей в зависимости от количества необходимых линий

# Неучтённые особенности моделирования
1. Рельсы и каретки надо вставлять как можно позже
2. Перфорацию имеет смысл делать как можно позже
