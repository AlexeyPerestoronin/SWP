# Pipe Clump Rubber Assembly Machine
Необходимо спроектировать автоматизированный механизм надевания уплотнительной резинки на сантехнических хомут.

## Ключевые требования:
1. Надевание резинки должно осуществляться НЕ методом проталкивания по длине хомута.
2. Процесс надевания резинки должен выполняться без смазки (она не должна быть необходимой).
3. Без использования электроники, только механика!

# Что можно улучшить в следующей версии:
1. Отсутствует какая-либо система продевания резинки на нужное расстояние с последующим отрезанием в заданных размерах.
2. Надо бы добавить какие-то ограничители, которые позволят регулировать силу с которой раздвижные губки давят на резинку в процессе её расширения (сейчас таковая сила равняется силе с которой толкатель прижимает хомут к головке упорной части).
3. Надо бы придумать механизм позволяющий регулировать пружинную силу, без замены самой пружины.
4. При движении толкателя линейные направляющие не должны выходить за пределы корпуса.
5. Необходимо стопоры, явно ограничивающие диапазон линейного перемещения для всех движущихся частей.
6. Надо постараться избежать выпирания линейных направляющих за пределы корпуса.
